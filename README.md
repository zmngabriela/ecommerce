# E-Commerce Project

Frontend e-commerce application built to explore product listing, authentication, state management, and shopping flow using modern React tools.

## Overview

This project was developed as a practical study in building a more complete application, including product browsing, authentication flow, form handling, and simulated order creation. It also helped me deepen my understanding of API integration, application state, and frontend architecture.

## Tech Stack

- React
- TypeScript
- Redux
- Styled Components
- Formik
- Yup
- Jest
- React Testing Library

## API

This project uses the Platzi Fake Store API for product and user data.

## Features

- Product listing and navigation
- JWT-based authentication flow
- Form validation
- Newsletter subscription
- Simulated order creation
- Pagination
- Responsive interface

## Challenges and Learning

Some parts of the experience had to be adapted because the API has limitations, especially around order creation and product variations. To work around this, I mocked some flows and focused on strengthening frontend logic, state handling, and authentication understanding.

## Future Improvements

- Improve product variation flow
- Expand test coverage
- Refine UI details
- Build a custom backend in Java in a future version
